Commands

Explore all the commands Excel has to offer

/help

Displays all available commands and their descriptions.

/help [command]

/J2C

Creates Temporary Vc.

/j2c setup

/ban

Bans a user from the server with an optional reason.

/ban [@user] [reason]

/kick

Kicks a user from the server with an optional reason.

/kick [@user] [reason]

/clear

Deletes a specified number of messages from the channel.

/clear [amount]

/poll

Creates a poll with multiple options for users to vote on.

/poll [question] [options]

/userinfo

Displays information about a specified user.

/userinfo [@user]

/serverinfo

Displays information about the current server.

/serverinfo

/afk

Sets your status as Away From Keyboard with a reason.

/afk [reason]
